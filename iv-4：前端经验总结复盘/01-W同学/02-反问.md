## 主流框架 Vue React Angular

- React 平均工资要高 —— 人基础能力强
- Vue 平均工资要低 —— “Vue 工程师”程序员基础能力要求并不强
- 国内 Vue 市场更大。Vue React 非常像，Vue 更能帮助你学 React（应用）
- React 更加简洁，一直用
