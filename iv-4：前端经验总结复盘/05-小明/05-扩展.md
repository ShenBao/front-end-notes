# 扩展

前端基础知识一般考察哪些方面？

## 理论基础

理论基础，即前端 HTML CSS JS HTTP 的基础知识，这些知识不一定能直接用到，但很重要。

例如
- CSS 盒子模型
- JS 原型、闭包、异步
- HTTP 状态码、cookie

## 基础应用

基础应用，即日常开发会经常用到的 API 和功能，这可以直接看出候选人的编码熟练度。

例如
- CSS 选择器优先级
- JS 数组 API
- JS DOM 操作 API
- 正则表达式

## 基础场景题

给一个项目功能和场景，让你设计出来，或者写出来，考察的也都是基础。

例如
- CSS 写一个抽屉
- JS 生成某一个 DOM 结构，如某种样子的 form 表单
- JS 设计一个 class 的继承
