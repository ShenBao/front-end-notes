# 传输安全

## HTTP 传输窃听

![191253.png](./img/191253.png)

traceroute 命令(unix) / tracert 命令(windows)
```bash
traceroute www.baidu.com

tracert www.baidu.com
```

![192209.png](./img/192209.png)

代理软件： anyproxy

危害：

- 窃取用户名密码
- 铭感信息
- 个人信息

HTTP 篡改：

- 插入广告
- 重定向网站
- 无法防御的 XSS 和 CSRF 攻击

场景：

- 运营商劫持
- 局域网劫持
- 公共wifi获取密码
