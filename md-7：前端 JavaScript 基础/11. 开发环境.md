# 开发环境

开发环境，最能体现工作产出效率

## 开发工具

- git
- chrome 调试工具
- 抓包（windows 一般用 fiddler, mac OS 一般用 charles）
- webpack
- babel
- linux 常用命令

webpack & babel
- ES6 模块化，浏览器暂不支持
- ES6 语法，浏览器并不完全支持
- 压缩代码，整合代码，已让网页加载更快

## linux基本命令

```bash
ls
ls -a
ll
mkdir aaa
cd aaa
pwd

rm -rf aaa
cp a.js aa.js
mv a.js  src/a.js
rm a.js

cat a.js
head a.js
tail a.js
head -a 1 a.js
tail -q 2 a.js

grep '2' a.js

vim a.js
i
esc
:w
:q
:!q
:wq
```
