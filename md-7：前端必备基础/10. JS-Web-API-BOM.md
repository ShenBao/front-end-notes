# JS-Web-API-BOM

BOM：Browser Object Model

问题

- 如何识别浏览器的类型
- 分析拆解 url 各个部分

知识点

- navigator
- screen
- location
- history

navigator

```js
const ua = navigator.userAgent;
const isChrome = ua.indexOf('Chrome');
console.log(isChrome);
```

screen

```js
screen.width;
screen.height;
```

## location

```js
location.href;
location.protocol;
location.pathname;
location.search;
location.hash;
```

## history

```js
history.back();
history.forward();
```
