## 为什么要学习 CSS 架构

- 日益膨胀的 CSS 代码，无计可施
- 项目长期处于无 CSS 架构维护状态
- 对 CSS 基础掌握过浅，难以发挥 CSS 威力

## 导致什么结果

- CSS 代码极度混乱
- CSS 难复用、难扩展、难维护
- 最终放弃治疗，推到重来吧

## 目标

- CSS 设计模式
- 掌握 CSS 架构，解决代码混乱问题
- CSS 奇淫巧技，发挥 CSS 威力
